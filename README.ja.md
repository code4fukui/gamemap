# ふくいゲームマップ

福井県のプログラミング好き小学生が考えた「もっと好きになる わたしのまち」とは!? - 全国選抜小学生プログラミング大会2020福井大会の作品を地図上に表示するアプリケーションです。

## デモ
https://code4fukui.github.io/gamemap/

## 機能
- 福井県の小学生が制作した作品の紹介を地図上で表示
- 作品の所属、学年、作品名、紹介文を吹き出しで表示
- ピン画像はデザイナーによりデザインされた可愛らしいものを使用

## 必要環境
ブラウザ上で動作します。

## 使い方
ブラウザで上記デモページを開くと、地図上に作品の位置が表示されます。ピンをクリックすると、その作品の詳細が表示されます。

## データ・API
- 作品一覧データ: [福井県大会作品一覧CSVオープンデータ](works-fukui2020.csv)
- 地図: [地理院地図](https://maps.gsi.go.jp/)
- ライブラリ: [Leaflet.js](https://leafletjs.com/)

# ふくいゲームマップ

福井県の小学生が制作した作品を地図上に表示するウェブアプリケーションです。

## 機能
- 福井県の地図上にマーカーで作品の位置を表示
- マーカーにはマウスオーバーすると作品名と所属情報が表示される
- オープンソースのLeaflet.jsライブラリを使用してマッピング機能を実現
- CSVファイルから作品の詳細データを読み込んで表示

## 必要環境
特別な環境は必要ありません。ウェブブラウザで動作します。

## 使い方
1. https://code4fukui.github.io/gamemap/ にアクセスする
2. 地図上にマーカーが表示されます
3. マーカーにマウスオーバーすると作品の詳細が表示されます

## データ・API
作品情報は`works-fukui2020.csv`ファイル、位置情報は`positions.csv`ファイルから取得しています。JavaScript側でデータを処理し、地図上に表示しています。
